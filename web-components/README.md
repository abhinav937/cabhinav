# Web Components Library

A collection of reusable web components for modern web development.

## Components

### Cube Progress Indicator
A 3x3 grid progress indicator arranged like a cube's face. Features smooth animations, interactive dots, and customizable progress levels.

**Features:**
- 3x3 grid of dots arranged like a cube face
- Smooth progress animations with easing
- Interactive dots (click to set progress)
- Slider control for manual progress setting
- Preset buttons for common progress levels
- Responsive design

**Demo:** [View Demo](cube-progress-indicator/index.html)

### Stat Dashboard
Animated stat cards dashboard perfect for displaying running statistics, fitness data, or any metrics. Features smooth number counting animations, hover effects, and progress bars.

**Features:**
- Animated number counting on load
- Multiple stat card types (distance, pace, time, elevation, heart rate, calories)
- Hover effects with glow and scale animations
- Progress bars for goal tracking
- Change indicators (positive/negative trends)
- Responsive grid layout
- Programmatic updates via JavaScript API
- Perfect for Strava running stats

**Demo:** [View Demo](stat-dashboard/index.html)

## Usage

Each component can be used independently. Simply copy the HTML, CSS, and JavaScript files to your project.

## Getting Started

1. Choose a component from the list above
2. Copy the component files to your project
3. Include the HTML structure, CSS styles, and JavaScript functionality
4. Customize as needed

## Component Structure

Each component follows this structure:
```
component-name/
├── index.html      # Demo and documentation
├── style.css       # Component styles
└── script.js       # Component functionality
```
