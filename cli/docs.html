Web Serial Terminal - Documentation

Getting Started

1. Click Connect button

2. Select your serial device

3. Configure baud rate in Settings if needed

4. Start sending commands

Browser Support

Chrome, Edge, or Opera (version 89+)

Firefox/Safari: Use Test Mode

Test Mode

Enable in Settings or add ?test=true to URL

Simulates serial communication without hardware

Commands

Basic: help info status ping time date version echo [text] uptime reset
Hardware: led [on|off|toggle] sensors read [sensor_name] gpio [pin] [value]

Keyboard Shortcuts

Enter - Send command

↑/↓ - Command history

Settings

Baud Rate: 9600, 115200, etc.

Data Bits: 7 or 8

Parity: none, even, odd

Stop Bits: 1 or 2

Show Timestamps: Toggle timestamps
